# Locus — Lofi Focus Timer

A minimalist Pomodoro timer with ambient soundscapes — customizable sessions, progress visualization, and keyboard shortcuts. Pure HTML/CSS/JS, Web Audio API, no dependencies.

**Live:** https://taras-polishchuk.github.io/locus/

---

## What it is

A side project I built for myself that turned into a good showcase of what's possible with vanilla JS and the Web Audio API — procedurally generated ambient sound, smooth UI transitions, and session state management without a single library.

---

## Features

- Pomodoro timer with customizable work/break durations
- Ambient soundscapes via Web Audio API (no audio files)
- Progress ring visualization
- Keyboard shortcuts
- Responsive, works on mobile

---

## Tech Stack

- HTML / CSS / JavaScript
- Web Audio API
